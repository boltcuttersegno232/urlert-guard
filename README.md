## ⚙️ What is urlert-guard?

urlert-guard helps you stay safe on the web. It checks websites in real time and warns you if a site looks risky. It works as a browser extension to scan domain names before you visit them. This way, you can avoid phishing scams and harmful web pages.

This tool runs right in your browser. It uses a client-side source code that processes URL information and uses threat intelligence to classify domains. You don't need to be a tech expert to use it — it is designed for everyday users who want simple protection while browsing.

1. **Download the extension package** from the release page.  
2. **Open your browser** (Chrome or Edge recommended).  
3. Navigate to the browser’s extension page:

4. Enable **Developer Mode** on the top right.  
5. Click **Load unpacked** or **Drag and drop** the `.crx` file if your browser allows.  
6. Select the downloaded extension folder or file.  
7. Confirm to add the extension to your browser.  
8. Once installed, look for the urlert-guard icon near the address bar.

The extension will now start scanning URLs you visit. It works silently in the background.

## 🔍 Using urlert-guard

After installation, urlert-guard monitors all websites you visit. It classifies domains in real time and flags suspicious ones.  

- A green check means the site is safe.  
- A yellow warning means “be careful.”  
- A red alert means do not proceed.

You can click the extension icon to see more details about any given domain. It shows data from threat intelligence sources and explains risks like phishing or malware.

If you visit a flagged site, urlert-guard blocks loading or shows a warning page depending on your settings.

The tool also updates its threat data automatically, so it stays current against new online threats.
